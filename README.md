# Elimu-Connect 🇰🇪

**Elimu-Connect** is a CBC-compliant (Competency-Based Curriculum) web portal designed to bridge the digital divide in underfunded rural schools in Kenya. Developed with a focus on accessibility and resource management, it provides a centralized platform for students, teachers, and administrators to manage educational content and school assets.

---

## 🚀 The Vision
In many rural schools, such as those on Mfangano Island, access to organized, curriculum-aligned digital resources is limited. Elimu-Connect is built to:
- **Digitize School Management:** Streamline the tracking of student assets (lockers, chairs, textbooks).
- **CBC Alignment:** Provide a portal that adheres to the Kenya Science and Engineering Fair standards and CBC requirements.
- **Data-Driven Insights:** Use data analytics to help school administrators understand resource allocation and student needs.

## 🛠️ Tech Stack
This project is built using a modern, scalable web stack:
- **Frontend:** HTML5, CSS3, EJS (Embedded JavaScript templates)
- **Backend:** Node.js, Express.js
- **Database:** MySQL (Relational database for robust data integrity)
- **Deployment & Version Control:** Git, GitHub

## ✨ Key Features
- **User Authentication:** Secure login/signup system for students and staff.
- **Asset Management:** A digitized ledger for tracking school furniture and equipment (Reduces loss and improves accountability).
- **Resource Library:** A digital repository for CBC-compliant learning materials.
- **Admin Dashboard:** A high-level view for school principals to monitor school health and inventory.

## 📈 Future Roadmap (Data Science Integration)
